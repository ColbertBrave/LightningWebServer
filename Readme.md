## 代码规范
+ 在满足需求的基础上，代码的清晰直白比使用各种复杂的特性更加重要；
+ 每个函数都要对传入参数的有效性进行检查；
+ 尽量避免使用C，不得不使用时，使用extern "C"将C语言的代码与C++代码进行分离编译，再统一链接；
